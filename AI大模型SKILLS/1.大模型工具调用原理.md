```
sequenceDiagram
    autonumber
    actor U as User
    participant C as ChatGPT Client
    participant G as API Gateway/Edge
    participant S as Conversation Service
    participant O as Orchestrator/Agent Runtime
    participant M as LLM Inference Service
    participant E as Tool Executor
    participant D as Conversation Store
    participant R as Retrieval/Search Index
    participant T as External Tools
    participant P as Safety/Policy Service

    U->>C: 输入问题
    C->>G: HTTPS请求
    G->>S: 转发请求
    S->>D: 读取历史/配置
    D-->>S: 返回上下文

    S->>P: 预检查
    P-->>S: 允许/拦截/降级

    S->>O: 创建执行上下文
    O->>M: 消息 + 工具定义 + 策略

    alt 可直接回答
        M-->>O: 文本回复
    else 需要工具
        M-->>O: tool call
        O->>E: 执行工具
        E->>T: 调用外部API/工具
        T-->>E: 工具结果
        E-->>O: 结构化结果

        opt 需要检索融合
            O->>R: 检索查询
            R-->>O: 检索片段
        end

        O->>M: 工具结果 + 检索结果
        M-->>O: 最终回答
    end

    O->>P: 后检查
    P-->>O: 通过/重写/拦截

    O->>D: 写入历史/元数据
    D-->>O: OK

    O-->>S: 最终结果
    S-->>G: 响应
    G-->>C: 流式/非流式输出
    C-->>U: 展示结果
```
